# 07_報工作業 v2 GAS 部署說明

## 目前定位

本階段改走 Apps Script 內建報工頁，不再由 GitHub Pages 直接送出報工資料。

原因：GitHub Pages 前端送出工號、產品、機台、良品、不良等資料時，容易遇到安全檢查與 CORS 限制。

## 已建立檔案

```text
07_報工作業v2_GAS入口.gs
```

已包含：

```text
開啟報工作業v2()
測試開啟報工作業v2入口()
```

## Apps Script 需新增 HTML 檔案

```text
07_報工作業v2.html
```

HTML 檔案內容由目前交付的 `work-report-v2.html` 作為基礎。

## 測試標準

1. Apps Script 下拉選單執行：

```text
測試開啟報工作業v2入口
```

應回傳：

```json
{
  "成功": true,
  "訊息": "報工作業 v2 入口函數已載入",
  "函數": "開啟報工作業v2"
}
```

2. Apps Script 下拉選單執行：

```text
開啟報工作業v2
```

應可開啟 HTML 報工頁。
