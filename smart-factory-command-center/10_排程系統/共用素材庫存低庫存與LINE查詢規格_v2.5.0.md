# 共用素材庫存、低庫存與 LINE 查詢規格 v2.5.0

## 一、目的

v2.5.0 將共用素材從「到貨與鎖料」推進到「日常維護、查詢、低庫存預警」。

核心能力：

```text
共用素材主檔維護
→ 安全庫存設定
→ 低庫存判斷
→ 鎖料歷史查詢
→ LINE 素材庫存查詢 / 低庫存圖卡
```

---

## 二、新增前端頁面

### 1. docs/material-master.html

共用素材主檔維護頁。

功能：

- 新增 / 更新共用素材主檔。
- 維護目前庫存、安全庫存、已鎖定數量、可分配數量。
- 維護主要供應商、採購前置天數、狀態。
- 篩選低庫存、可分配不足、啟用、停用。
- 匯出 CSV。

### 2. docs/lock-history.html

鎖料分配歷史查詢頁。

功能：

- 依日期區間查詢。
- 依分配狀態查詢：已滿足、部分滿足、未分配。
- 搜尋分配批號、共用素材、需求編號、產品、經辦人。
- 匯出 CSV。
- 產生並複製鎖料歷史摘要。

```text

```

新增函數：

```text

產生LINE_素材庫存文字_(關鍵字)

產生LINE_低庫存Flex_()
產生LINE回覆訊息_v2_5_(文字)
```

---

## 四、低庫存判斷規則

| 判斷 | 條件 |
|---|---|
| 低庫存 | 目前庫存 <= 安全庫存 |
| 缺安全庫存 | 安全庫存空白 |
| 可分配不足 | 可分配數量 <= 0 |

---

## 五、LINE 指令

| 指令 | 回覆 |
|---|---|
| 低庫存 | LINE 低庫存 Flex 圖卡 |
| 低庫存圖卡 | LINE 低庫存 Flex 圖卡 |
| 素材庫存 關鍵字 | 文字版素材庫存查詢 |
| 素材低庫存 | 文字版低庫存摘要 |

範例：

```text
素材庫存 A801910008
素材庫存 棧板
低庫存
```

## 六、Webhook 入口

若要啟用 v2.5 最新 LINE 指令，Webhook 回覆入口需改為：

```text
產生LINE回覆訊息_v2_5_(文字)
```
