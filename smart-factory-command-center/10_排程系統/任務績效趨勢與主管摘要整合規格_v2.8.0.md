# 任務績效趨勢與主管摘要整合規格 v2.8.0

## 一、目的

v2.8.0 將三方待辦任務從「可統計」推進到「可追蹤績效、可納入主管會議、可推播逾期」。

核心流程：

```text
跨部門待辦任務
→ 負責人績效統計
→ 任務完成率趨勢
→ 任務逾期推播
→ 主管摘要整合任務段落
```

---

## 二、新增前端頁面

```text
docs/task-performance.html
```

功能：

- 負責人績效統計。
- 任務完成率趨勢圖。
- 任務逾期趨勢圖。
- 負責人績效 CSV 下載。
- 完成率趨勢 CSV 下載。
- 績效摘要複製。

---

## 三、主管摘要頁升級

更新頁面：

```text
docs/meeting.html
```

新增能力：

- 主管摘要合併跨部門任務週報。
- 可單獨產生任務段落。
- 主管摘要增加第九段：跨部門任務追蹤。
- 顯示任務總數、未完成、已完成、逾期、2天內到期、高優先。
- 顯示負責人績效 TOP。
- 顯示高風險任務清單。

---

## 五、負責人績效統計

依 `10_跨部門待辦任務` 的 `負責人` 分組：

```text
任務總數
已完成
未完成
逾期
高優先
完成率
```

完成率：

```text
完成率 = 已完成 / 任務總數 * 100%
```

---

## 六、完成率趨勢

依任務建立日期分組：

```text
日期
任務數
完成數
逾期數
完成率
```

用途：

- 追蹤每日任務完成狀況。
- 判斷跨部門任務是否累積未結。
- 將任務改善成果轉為可視化趨勢。

## 七、LINE 指令

| 指令 | 回覆 |
|---|---|
| 負責人績效 | 文字版負責人績效 |
| 任務績效 | 文字版負責人績效 |
| 任務逾期 | 文字版逾期任務預覽 |
| 逾期任務 | 文字版逾期任務預覽 |
| 主管任務摘要 | 主管摘要任務段落 |

Webhook 若要啟用 v2.8 指令：

```text
產生LINE回覆訊息_v2_8_(文字)
```

---

## 八、逾期推播

函數：

```text
推播_v2_8_LINE任務逾期清單(目標ID)
```

注意：

- 未提供目標ID時，只回傳預覽，不推播。
- 需設定 `LINE_CHANNEL_ACCESS_TOKEN` 才能正式推播。
