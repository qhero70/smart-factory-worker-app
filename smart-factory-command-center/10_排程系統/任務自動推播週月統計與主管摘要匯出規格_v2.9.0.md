# 任務自動推播、週月統計與主管摘要匯出規格 v2.9.0

## 一、目的

v2.9.0 將任務追蹤從「手動查詢」升級為「每日自動提醒與會議資料留存」。

核心流程：

```text
任務資料
→ 每日任務摘要推播
→ 逾期任務推播
→ 週/月完成率統計
→ 主管摘要 PDF / CSV 匯出
```

---

## 二、新增 GAS 檔案

```text
01_GAS後端/v2_9_任務自動推播週月統計與摘要匯出.gs
```

新增函數：

```text
升級_v2_9_任務自動推播設定表()
設定_v2_9_任務自動推播(資料)
建立_v2_9_每日任務摘要觸發器()
每日執行_v2_9_任務摘要推播()
每日執行_v2_9_任務逾期推播()
取得_v2_9_任務週月完成率統計()
匯出_v2_9_主管摘要CSV(摘要文字)
匯出_v2_9_主管摘要PDF(摘要文字)
```

---

## 三、新增資料表

```text
00_任務自動推播設定
```

欄位：

```text
設定項目、設定值、備註、更新時間
```

預設設定：

```text
LINE任務推播目標ID
每日任務摘要推播小時
每日任務摘要啟用
任務逾期推播啟用
```

---

## 四、新增前端頁面

```text
docs/task-auto-push.html
```

功能：

- 設定 LINE 任務推播目標 ID。
- 設定每日推播小時。
- 啟用 / 停用每日任務摘要。
- 啟用 / 停用任務逾期推播。
- 儲存後建立 GAS 觸發器。
- 預覽任務週報。
- 預覽逾期任務。
- 查詢週 / 月完成率統計。

---

## 五、主管摘要頁升級

更新：

```text
docs/meeting.html
```

新增：

- 匯出 CSV。
- 匯出 PDF。
- 下載 TXT。
- 匯出檔案會建立於 Google Drive。

---

## 六、使用前必執行

```text
升級_v2_9_任務自動推播設定表()
```

若要啟用每日推播，需：

```text
1. 設定 LINE_CHANNEL_ACCESS_TOKEN
2. 填入 LINE任務推播目標ID
3. 每日任務摘要啟用 = 是
4. 任務逾期推播啟用 = 是
```
