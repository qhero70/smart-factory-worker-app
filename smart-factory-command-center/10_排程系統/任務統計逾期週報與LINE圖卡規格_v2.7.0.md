# 任務統計、逾期週報與 LINE 圖卡規格 v2.7.0

## 一、目的

v2.7.0 將三方待辦任務從「狀態看板」升級為「主管追蹤儀表板」。

核心能力：

```text
三方待辦任務
→ 任務統計
→ 逾期警示
→ 2 天內到期提醒
→ 任務週報
→ LINE 任務摘要 Flex 圖卡
```

---

## 二、新增前端頁面

```text
docs/task-dashboard.html
```

功能：

- 任務總數。
- 未完成。
- 已完成。
- 逾期。
- 2 天內到期。
- 高優先。
- 負責單位統計。
- 任務狀態統計。
- 優先級統計。
- 高風險任務清單。
- 任務週報產出與複製。
- 高風險任務 CSV 下載。

## 四、逾期判斷

任務狀態不是「完成」，且：

```text
最晚採購日 < 今日
```

即為逾期。

---

## 五、即將逾期判斷

任務狀態不是「完成」，且：

```text
今日 <= 最晚採購日 <= 今日 + 2天
```

即為 2 天內到期。
