# 版本更新紀錄

## v1.0.0｜正式主線建立

- 建立 `smart-factory-command-center/` 正式專案目錄。
- 建立 `docs/` GitHub Pages 入口。
- 建立 GAS 後端、HTML5 PWA 前端與 LINE Bot 單一入口設定。
- 建立部署說明、測試資料、欄位規格、驗收清單。

## v1.1.0｜系統維護與部署穩定化

- 新增 `.nojekyll`、`appsscript.json`、`系統維護工具.gs`。
- 支援一鍵自檢、欄位修復、建立備份、CSV 匯入、每日戰情快照。

## v1.2.0｜前端戰情室強化與維護工具接入

- 新增 KPI、系統自檢、欄位修復、備份、每日快照、測試資料匯入 API。
- `docs/app.html` 新增 KPI、維護功能與圖表。
- LINE Bot 戰情回覆增加達成率、不良率、停機分鐘、逾期警示。

## v1.3.0｜正式資料管理與資料治理

- 新增主檔管理 API：取得、新增/更新、停用、CSV 匯入。
- 新增 GitHub Pages 資料管理頁：`docs/data.html`。
- 新增正式資料 CSV 模板與資料治理規則。

## v1.4.0｜生產計畫清洗、工單生成、排程需求池
