# v3.0.0 正式部署候選版規格

## 一、版本定位

```text
v3.0.0 = 正式部署候選版
```

此版目的不是再新增單一功能，而是將 v1.0 ~ v2.9 已完成的功能整併成可部署、可驗收、可交付的正式系統。

---

## 二、本版完成項目

### 1. GAS 主控入口整併

新增：

```text
01_GAS後端/v3_0_GAS主控入口與LINE最終整合版.gs
```

功能：

- 統一 `doGet(e)`。
- 統一 `doPost(e)`。
- 支援 GitHub Pages 前端 API。
- 支援 LINE Webhook。
- 支援健康檢查。
- 支援正式上線初始化。
- 支援系統總檢查。
- 向下相容 v2.8 / v2.7 / v2.5 / v2.4 / 舊 LINE 指令。

---

### 2. LINE Webhook 最終入口

正式入口：

```text
產生LINE回覆訊息_v3_0_(文字)
```

新增 LINE 指令：

```text
版本
系統版本
健康檢查
系統狀態
總檢查
上線檢查
```

---

### 3. 全頁連線測試

確認頁面：

```text
docs/system-test.html
```

測試範圍：

- GitHub Pages 全頁是否可開啟。
- GAS Web App URL 是否可連線。
- 健康檢查是否成功。
- 核心 API 是否存在。
- v3.0 系統總檢查是否成功。

---

### 4. 正式部署交付文件

新增：

```text
04_部署說明/v3_0_正式部署交付文件.md
```

內容：

- 系統定位。
- Apps Script 貼入順序。
- 初始化步驟。
- GAS Web App 部署設定。
- LINE Webhook 設定。
- GitHub Pages 頁面清單。
- 正式上線測試。
- 驗收標準。

---

### 5. 首頁升級

更新：

```text
docs/index.html
```

新增：

- v3.0.0 正式部署候選版標示。
- 全頁測試入口。
- 正式部署前提醒。

---

## 三、正式上線前必執行

Apps Script 執行：

```text
初始化_v3_0_正式上線()
```

LINE Webhook 使用 GAS Web App URL。

前端測試使用：

```text
docs/system-test.html
```
