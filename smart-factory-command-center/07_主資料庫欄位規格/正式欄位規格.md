# 正式欄位規格

## 00_系統設定

| 欄位 | 說明 |
|---|---|
| 項目 | 設定項目名稱 |
| 數值 | 設定值 |
| 備註 | 補充說明 |
| 最後更新時間 | 更新時間 |

## 01_人員主檔

| 欄位 | 說明 |
|---|---|
| 工號 | 員工唯一編號 |
| 姓名 | 中文姓名 |
| 部門 | 所屬部門 |
| 組別 | 製造一組、清洗區等 |
| 職稱 | 工程師、技術員、作業員 |
| 班別 | 早班、中班、夜班 |
| LINE_USER_ID | LINE 綁定 ID |
| 啟用 | 是 / 否 |
| 備註 | 補充 |
| 更新時間 | 最後更新時間 |

## 02_產品主檔

| 欄位 | 說明 |
|---|---|
| 產品編號 | 產品料號 |
| 客戶品號 | 客戶端品號 |
| 品名 | 產品名稱 |
| 分類 | 成品 / 素材 / 半成品 |
| 標準工時_秒 | 單件標準工時 |
| 標準產能_班 | 每班標準產能 |
| 啟用 | 是 / 否 |
| 備註 | 補充 |
| 更新時間 | 最後更新時間 |

分類規則：

- 末碼 0：成品
- 末碼 1：素材
- 末碼 8：半成品 / 待組 / 待精加工

## 03_機台主檔

| 欄位 | 說明 |
|---|---|
| 機台編號 | 現場機台號 |
| 機台名稱 | 顯示名稱 |
| 區域 | A1~A9 等 |
| 組別 | 負責組別 |
| 型號 | 機型 |
| 加工材質 | 鐵 / 鋁 / 鋼 |
| 另裝配備 | 分度盤、內藏探頭、自動換刀裝置等 |
| 狀態 | 可生產 / 保養 / 停用 / 報廢 |
| 啟用 | 是 / 否 |
| 更新時間 | 最後更新時間 |

切削液濃度基準：

- 鐵：2~4%
- 鋁：5~7%
- 鋼：7~9%

## 09_報工紀錄

報工主表，所有現場產出紀錄都寫入此表。

重要欄位：

- 作業日：依 06:10 前算前一日規則。
- 良品數：本次完成良品。
- 不良數：本次不良。
- 停機分鐘：同次報工附帶停機分鐘。
- 換刀次數：同次報工附帶換刀次數。
- 來源：HTML5_PWA / LINEBot / 後台匯入。
- 狀態：有效 / 已撤銷。

## 10_工單主檔

工單狀態：

- 待生產
- 進行中
- 已完成
- 暫停
- 取消

新增報工後，系統會依工單編號自動累加：

- 已完成量
- 不良量
- 狀態
- 更新時間

## 11_檢具主檔

未來檢具系統使用：

```text
產品 → 箱號 → 檢具 → 位置 → 狀態 → 校驗週期
```

## 12_AI分析紀錄

用於保存 AI 分析結果，包含：

- 戰情摘要
- 交期風險
- 達成率分析
- 建議動作
- 風險等級
