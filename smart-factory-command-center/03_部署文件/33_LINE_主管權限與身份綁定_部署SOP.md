# 33_LINE｜主管權限與身份綁定｜部署 SOP

## 一、模組定位

第 33 階段只處理 LINE 身份綁定與權限控管，不改 Rich Menu 圖片，不改 PWA，不進入排程。

目的：

- 誰可以看主管戰情。
- 誰可以查今日戰情 / 昨日戰情。
- 誰可以查主檔檢查。
- 誰可以看 AI摘要。
- 現場人員保持可報工。

## 二、已新增 GAS 檔案

```text
smart-factory-command-center/01_GAS後端/33_LINE_主管權限與身份綁定.gs
```

## 三、會自動建立的工作表

| 工作表 | 用途 |
|---|---|
| 33_LINE身份權限 | LINE_USER_ID、工號、姓名、角色、權限欄位 |
| 33_LINE權限紀錄 | 每次綁定、查詢、攔截、放行紀錄 |

## 四、主後端接線位置

在 `智慧製造中央作戰指揮中心.gs` 的 `doPost(e)` 裡，找到這一段：

```javascript
if (p && p.events && Array.isArray(p.events)) {
```

在它下一行立刻插入：

```javascript
    if (typeof LINE身份權限_嘗試處理Webhook_ === 'function') {
      var 身份權限結果 = LINE身份權限_嘗試處理Webhook_(p);
      if (身份權限結果 && 身份權限結果.已處理) return 主程式_安全輸出JSON_(身份權限結果);
    }
```

插入後的正確樣子：

```javascript
  if (p && p.events && Array.isArray(p.events)) {
    if (typeof LINE身份權限_嘗試處理Webhook_ === 'function') {
      var 身份權限結果 = LINE身份權限_嘗試處理Webhook_(p);
      if (身份權限結果 && 身份權限結果.已處理) return 主程式_安全輸出JSON_(身份權限結果);
    }

    if (typeof LINE主管戰情日期快選_嘗試處理Webhook_ === 'function') {
      var 日期快選結果 = LINE主管戰情日期快選_嘗試處理Webhook_(p);
      if (日期快選結果 && 日期快選結果.已處理) return 主程式_安全輸出JSON_(日期快選結果);
    }
```

## 五、部署順序

1. 將 `33_LINE_主管權限與身份綁定.gs` 加入 Apps Script 專案。
2. 在主後端 `doPost(e)` 接入上方 5 行權限閘門。
3. 執行：

```javascript
初始化33_LINE主管權限與身份綁定()
```

4. 執行：

```javascript
測試_33_LINE主管權限與身份綁定_本機規格()
```

5. 執行：

```javascript
測試_33_LINE主管權限_模擬未綁定主管指令()
```

## 六、LINE 使用方式

### 綁定身份

```text
綁定 工號
```

範例：

```text
綁定 fhfi573
```

### 查詢自己身份

```text
權限檢查
```

或：

```text
我的身份
```

### 解除綁定

```text
解除綁定
```

## 七、預設權限規則

| 角色 | 主管入口 | 主檔檢查 | AI摘要 | 報工作業 |
|---|---|---|---|---|
| 主管 | 是 | 是 | 是 | 是 |
| 班長 | 是 | 是 | 是 | 是 |
| 工程師 | 是 | 是 | 是 | 是 |
| 現場人員 | 否 | 否 | 否 | 是 |

## 八、保護指令

| 指令 | 權限欄位 |
|---|---|
| 主管戰情 | 允許主管入口 |
| 今日戰情 | 允許主管入口 |
| 昨日戰情 | 允許主管入口 |
| 戰情 yyyy-mm-dd | 允許主管入口 |
| 主檔檢查 | 允許主檔檢查 |
| AI摘要 | 允許AI摘要 |

## 九、驗收標準

| 測試項目 | 成功標準 |
|---|---|
| 未綁定者點主管戰情 | 回覆請先綁定身份 |
| 已綁定主管點主管戰情 | 正常回覆主管戰情 |
| 現場人員點主管戰情 | 回覆權限不足 |
| 現場人員點報工作業 | 正常回覆報工入口 |
| 權限檢查 | 回覆姓名、工號、角色、權限 |
| Google Sheets | 出現 33_LINE身份權限、33_LINE權限紀錄 |

## 十、標準

- 只接 LINE webhook 權限閘門。
- 不改 Rich Menu。
- 不改報工頁。
- 不改排程。
- 不破壞既有 LINE 文字指令。
- 現場人員仍可報工。
