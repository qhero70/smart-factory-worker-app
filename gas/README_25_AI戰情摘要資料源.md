# 25_AI戰情摘要資料源

本階段目的：將 24_派班報工每日結算、未報工追蹤、派班效率統計、巡檢異常明細整理成 AI / LINE / 主管戰情可讀資料源。

## 正式檔案

```text
gas/25_AI戰情摘要資料源正式模組.gs
gas/00_主程式_doPost_AI戰情接入版_v25.gs
```

## API Action

```text
AI戰情資料源健康檢查
產生AI戰情摘要資料源
取得AI戰情摘要資料源
```

## 輸入資料來源

```text
24_派班報工日結
24_未報工追蹤
24_派班效率統計
23_派班報工異常明細
```

## 自動建立分頁

```text
25_AI戰情摘要資料源
25_AI戰情風險清單
25_LINE每日摘要佇列
25_AI戰情摘要紀錄
```

## 輸出內容

```text
風險等級
派班筆數
已報工派班數
部分報工派班數
未報工派班數
派工量合計
良品數
不良數
報工數

不良率
AI摘要文字
主管建議
LINE摘要文字
```

## Apps Script 測試函數

```javascript
測試_AI戰情資料源健康檢查()

測試_取得AI戰情摘要資料源()
```

## 發佈步驟

```text
1. Apps Script 新增 25_AI戰情摘要資料源正式模組.gs
2. 主後端替換 doPost 接入版：00_主程式_doPost_AI戰情接入版_v25.gs
3. 儲存
4. 部署 → 管理部署作業 → 編輯 Web App → 新增版本 → 部署
5. 先測 AI戰情資料源健康檢查
6. 執行 產生AI戰情摘要資料源
7. 檢查 25_AI戰情摘要資料源 / 25_AI戰情風險清單 / 25_LINE每日摘要佇列
```
