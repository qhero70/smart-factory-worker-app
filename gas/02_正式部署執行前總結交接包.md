## 十二、回滾點

### PWA 回滾

可回到 GitHub 前一 commit 或把入口版本退回 `v267 / v268` 對應穩定版。

### GAS 回滾

1. Apps Script 管理部署。
2. 選回上一個穩定版本。
3. 不刪 Google Sheets 資料。
4. 不刪 `33_LINE身份權限`。
5. 若 LINE 綁定異常，只把 H 欄啟用改為 `否`。

### LINE 綁定錯人

1. 到 `33_LINE身份權限` 找工號。
2. H欄改為 `否`。
3. I欄填 `人工停用：綁定錯誤`。
4. 正確人員重新輸入 `綁定 工號`。
