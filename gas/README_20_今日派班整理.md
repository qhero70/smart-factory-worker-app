# 20_今日派班整理正式模組

階段：10_每日派班表 → 10_今日派班表 → PWA 報工入口

## 目的

`10_每日派班表` 會依建議開工日排序，因此逾期工單可能出現過去日期。  
現場派班不能直接拿過去日期作業，所以本模組會把逾期派班轉成今日補派。

## 模組檔案

```text
gas/20_今日派班整理正式模組.gs
```

## 主要函數

```javascript
今日派班_產生今日表(payload)
測試_產生今日派班表()
```

## 輸入來源

```text
10_每日派班表
```

## 輸出目標

```text
10_今日派班表
10_今日派班整理紀錄
```

## 日期規則

```text
原派班日期 < 今日 → 日期狀態 = 逾期補派，作業日改為今日
原派班日期 = 今日 → 日期狀態 = 今日
原派班日期 > 今日 → 預設不納入今日表
```

## 班別時間規則

```text
早班：08:00 ~ 16:50
加班：17:20 ~ 20:20
大夜班：23:00 ~ 隔日 07:50
```

## 使用方式

執行：

```javascript
測試_產生今日派班表()
```

成功後檢查：

```text
10_今日派班表
10_今日派班整理紀錄
```

## 成功標準

`10_今日派班表` 應包含：

```text
今日派班編號
作業日
原派班日期
日期狀態
班別
工號
姓名
工站
工序
工序明細編號
工單編號
產品編號
品名
主機台
派工量
報工狀態
現場確認
```

## 下一階段

今日派班表完成後，下一段進入：

```text
10_今日派班表
↓
PWA 報工入口帶入今日派班
↓
報工扣工單 / 扣工序
```
